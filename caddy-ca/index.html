🔒 Instalar Certificado UNESPAR

Necessário para acessar o Diário de Bordo com segurança.

⬇ Baixar certificado
Android: Baixe → Configurações → Segurança → Instalar certificado → CA
iPhone: Baixe → Configurações → Perfil Baixado → Instalar → Configurações → Geral → Sobre → Confiança de Certificados → ativar

🚀 Abrir o App
